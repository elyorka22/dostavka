# Папка для изображений

Эта папка предназначена для хранения изображений категорий и товаров.

## Структура папок:

- `categories/` - изображения для категорий
- `products/` - изображения для товаров

## Как загрузить изображение:

### Вариант 1: Через админ-панель (рекомендуется)
1. Войдите как супер-админ
2. Перейдите в "Kategoriyalarni boshqarish"
3. Нажмите "Rasm yuklash" для нужной категории
4. Выберите изображение
5. Изображение будет сохранено автоматически

### Вариант 2: Вручную
1. Скопируйте изображение в соответствующую папку (`categories/` или `products/`)
2. Назовите файл понятным именем (например: `vegetables.jpg`, `fruits.png`)
3. В админ-панели укажите путь к файлу (например: `images/categories/vegetables.jpg`)

## Поддерживаемые форматы:
- JPEG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)
- GIF (.gif)

## Рекомендуемый размер:
- Для категорий: 200x200px или больше (квадратные)
- Для товаров: 400x400px или больше (квадратные)
- Максимальный размер файла: 2MB
